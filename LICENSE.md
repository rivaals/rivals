## 🔧 Common Issues & Fixes

### ❌ Script Loads But GUI Doesn't Appear
**Cause:** Script executed before game fully loaded  
**Fix:** Wait until your character is fully spawned and you can move, then re-execute the loader. Do not inject during the Roblox loading screen.

### ❌ Silent Aim Not Working
**Cause:** Target prediction disabled or FOV set to 0  
**Fix:** Open the Aim tab → ensure Silent Aim is toggled ON → set FOV to at least 120 → set Target Priority to "Closest Enemy." If still not working, re-execute the script after re-injecting.
